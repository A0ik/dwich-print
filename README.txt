========================================
  DWICH62 - Serveur d'impression
========================================

INSTALLATION:
1. Installez Node.js depuis https://nodejs.org/ (version LTS)
2. Double-cliquez sur "install.bat"
3. C'est pret!

UTILISATION:
- Double-cliquez sur "start.bat" pour demarrer
- Le serveur doit rester ouvert pour imprimer

DEMARRAGE AUTOMATIQUE (optionnel):
Pour que le serveur demarre avec Windows:
1. Appuyez sur Win+R
2. Tapez: shell:startup
3. Copiez "start-hidden.vbs" dans ce dossier

TEST:
- Ouvrez http://localhost:3333/test dans votre navigateur
- 2 tickets de test devraient s'imprimer

CONFIGURATION:
- Nom imprimante: AURES ODP333 (modifiable dans server.js)
- Port: 3333
